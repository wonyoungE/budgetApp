# budget-cli

## 목적
이 스킬은 CSV 기반 Python CLI 가계부 앱을 개발할 때 사용하는 프로젝트 전용 스킬이다.

## 적용 시점
- 거래 추가, 잔액 계산, 카테고리 필터링, 월별 요약, CSV 로드 기능을 작업할 때 사용한다.
- 테스트 우선 개발과 품질 검수를 반복해야 할 때 사용한다.

## 작업 원칙
- 먼저 테스트를 작성한다.
- 구현은 작은 단위로 나눈다.
- 모든 함수에 타입 힌트를 붙인다.
- 함수 길이는 50줄 이하로 유지한다.
- 순환복잡도 10 이하를 유지한다.

## 검증 명령
- `pytest`
- `radon cc`

## 완료 기준
- 기능 테스트가 추가되었고 통과한다.
- `radon cc` 기준에서 순환복잡도 위반이 없다.
- `qa_engineer` 서브에이전트 검수를 통과한다.
